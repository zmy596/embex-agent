# LED 与无源蜂鸣器开发规则

用途：指导 Embex 生成基础执行器控制代码。

## LED

LED 控制需要确认：

- GPIO 编号。
- 高电平有效还是低电平有效。
- 是否经过三极管、MOS 管或模块驱动。
- 是否需要 PWM 呼吸灯。

常见行为：

- 常亮。
- 熄灭。
- 固定周期闪烁。
- PWM 呼吸灯。
- 状态/告警指示。

验收：

- 串口输出 LED 引脚、有效电平和状态变化。
- 用户观察 LED 行为是否与任务一致。

## 无源蜂鸣器

无源蜂鸣器需要 PWM 或 `tone()` 输出频率，不是简单高低电平持续输出。

注意：

- 有源蜂鸣器和无源蜂鸣器控制方式不同。
- 蜂鸣器建议使用三极管驱动，避免 GPIO 直接带大负载。
- 播放旋律时避免长时间阻塞；如果必须阻塞，应保持串口日志说明当前状态。

验收：

- 串口输出 `[BUZZER]` 事件、频率或旋律名。
- 用户确认蜂鸣器发声。
- 若不响，检查有源/无源类型、电源、驱动三极管、GPIO、电平有效性。
